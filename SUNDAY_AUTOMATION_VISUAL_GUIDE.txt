🪟 WINDOWS USERS

   OPTION 1: Automatic Setup (Recommended)
   ────────────────────────────────────────
   1. Right-click "schedule-sunday-startup.bat"
   2. Select "Run as administrator"
   3. Follow the prompts
   
   ✅ Done! Application will start every Sunday at 7:00 AM

   OPTION 2: Manual Test
   ─────────────────────
   Double-click: sunday-auto-start.bat
   
   OPTION 3: Windows Service (24/7 Running)
   ────────────────────────────────────────
   See AUTOMATION_GUIDE.md for NSSM setup


🍎 MAC USERS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   OPTION 1: Automatic Cron Setup
   ───────────────────────────────
   Open Terminal and run:
   
   cd /path/to/ProgramGenerator
   chmod +x setup-sunday-cron.sh
   ./setup-sunday-cron.sh
   
   ✅ Done! Application will start every Sunday at 7:00 AM

   OPTION 2: Manual Test
   ─────────────────────
   bash sunday-auto-start.sh


🐧 LINUX USERS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   OPTION 1: Automatic Cron Setup
   ───────────────────────────────
   Open Terminal and run:
   
   cd /path/to/ProgramGenerator
   chmod +x setup-sunday-cron.sh
   ./setup-sunday-cron.sh
   
   ✅ Done! Application will start every Sunday at 7:00 AM

   OPTION 2: Manual Cron Edit
   ──────────────────────────
   crontab -e
   
   Add: 0 7 * * 0 cd /path/to/ProgramGenerator && bash sunday-auto-start.sh

┌────────────────────────────────────────────────────────────────┐
│  🆘 TROUBLESHOOTING                                            │
└────────────────────────────────────────────────────────────────┘

  ❌ Application doesn't start?
  ────────────────────────────
  • Check Java is installed: java -version
  • Verify file paths in scheduled task
  • Run script manually to test
  
  ❌ Browser doesn't open?
  ───────────────────────
  • Manually open: http://localhost:8080/sacrament
  • Check if port 8080 is free
  • Verify application started successfully
  
  ❌ Scheduled task not running?
  ─────────────────────────────
  Windows: Check Task Scheduler history
  Linux/Mac: Check cron logs (grep CRON /var/log/syslog)
