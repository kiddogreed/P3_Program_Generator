# Church Program Generator - Ready to Deploy 
 
This folder contains everything needed to run the Church Program Generator. 
 
For Windows: 
1. Double-click "run-church-program.bat" 
2. Open browser to http://localhost:8080 
 
For Mac/Linux: 
1. Run "bash run-church-program.sh" 
2. Open browser to http://localhost:8080 
 
Requirements: Java 17 or higher
