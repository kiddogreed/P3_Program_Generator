This directory contains all generated Bishopric Meeting agendas.

Files are automatically saved here with the naming format:
bishopricMeeting{YYYY-MM-DD}.docx
bishopricMeeting{YYYY-MM-DD}.pdf

Example files:
- bishopricMeeting2025-10-12.docx
- bishopricMeeting2025-10-12.pdf
- bishopricMeeting2025-10-15.docx
- bishopricMeeting2025-10-15.pdf

Generated automatically when exporting Bishopric meeting agendas.
