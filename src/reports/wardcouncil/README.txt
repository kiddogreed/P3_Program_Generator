This directory contains all generated Ward Council Meeting agendas.

Files are automatically saved here with the naming format:
wardCouncilMeeting{YYYY-MM-DD}.docx
wardCouncilMeeting{YYYY-MM-DD}.pdf

Example files:
- wardCouncilMeeting2025-10-12.docx
- wardCouncilMeeting2025-10-12.pdf
- wardCouncilMeeting2025-11-02.docx
- wardCouncilMeeting2025-11-02.pdf

Generated automatically when exporting Ward Council meeting agendas.
